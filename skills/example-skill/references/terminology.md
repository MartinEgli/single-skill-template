# Terminology

Use consistent terms:

- Skill: an agent instruction package with `SKILL.md`.
- Reference: supporting method or context read when relevant.
- Asset: reusable output structure or template.
- Example: realistic sample input or output proving expected behavior.
- Mode: named workflow within the skill.
- Quality gate: check that must pass before final output.

Do not create domain-specific abbreviations in this generic template.
