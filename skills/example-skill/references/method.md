# Method

Use this sequence for substantial work:

1. Identify goal and user outcome.
2. Extract constraints and source facts.
3. Choose mode: plan, review, build, or final-check.
4. Apply relevant assets only when structure helps.
5. Separate evidence, inference, assumptions, and gaps.
6. End with concise next action or completed artifact.

Keep method lightweight. The skill should make work clearer, not add ceremony.
