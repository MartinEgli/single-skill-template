# Evidence Discipline

Use four labels when accuracy matters:

Evidence:
Directly supported by supplied material or verified source.

Inference:
Reasoned conclusion from evidence. Mark it as inference when not directly
stated.

Assumption:
Useful working belief that has not been verified.

Gap:
Missing information that could change the result.

Rules:

- Do not upgrade assumptions into facts.
- Do not cite sources that were not inspected.
- Prefer exact file paths, line references, commands, and identifiers when
  available.
- If current external information matters, verify it before relying on memory.
