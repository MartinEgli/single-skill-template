# Example Skill

This skill is the canonical example for `single-skill-template`.

It demonstrates:

- required `SKILL.md` structure
- reference files for supporting method
- assets for output contracts
- examples for expected behavior

## Modes

- `/example plan`
- `/example review`
- `/example build`
- `/example final-check`

## Customize

When deriving a real skill:

1. Rename the folder.
2. Update frontmatter `name`.
3. Replace generic references with domain method.
4. Replace examples with realistic user tasks.
5. Run `npm run validate`.
